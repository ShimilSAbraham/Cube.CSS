# Cube.CSS

#### To use this in your html, type in the following code:
```
<div class="cube">
  <div class="front grey-l"></div>
  <div class="side grey-d"></div>
  <div class="top grey"></div>
</div>
```
#### Colors that are available:
* purple
* pink
* blue
* red
* green
* orange
* indigo
* grey
* pacha
* yellow
